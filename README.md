# Pewdys Cafe ☁️

A cosy coffee and tea café website built with love.

**Live site:** [View Website](https://yourusername.github.io/pewdys-cafe)

## About
Pewdys Cafe is a modern café specialising in coffee, tea and weekend desserts. 
Built as a single-page HTML website with no frameworks required.

## Files
- `index.html` — the entire website (just this one file!)

## Features
- Full drink menu (Coffee, Non-Coffee, Hot Tea with sub-categories)
- All day brunch menu
- Weekend dessert night menu
- Fully mobile responsive
- Custom animated cursor
